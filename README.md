## 🆕 行程 v2 修訂版（最終版）

### 路線變更

從原本的「華沙 2 + 克拉科夫 5（含 Zakopane / 鹽礦 / Auschwitz 三個一日往返）」升級為四城深度路線：

| 城市 | 晚數 | 主要行程 |
|---|---|---|
| 🇵🇱 **華沙 Warszawa** | 1 晚（Day 1） | 抵達日老城散步、倒時差 |
| 🇵🇱 **克拉科夫 Kraków** | 3 晚（Day 2-4） | 老城三大景點 + Auschwitz + 辛德勒工廠 + Kazimierz |
| 🇵🇱 **弗羅茨瓦夫 Wrocław** | 1 晚（Day 5） | 700 小矮人 + 百年廳 + 全景畫 + 座堂島煤氣燈點燈 |
| 🇵🇱 **波茲南 Poznań** | 1 晚（Day 6） | 教堂島 + 12:00 山羊鐘樓秀 + 聖馬丁牛角麵包 |
| 🇵🇱 **華沙 Warszawa** | 1 晚（Day 7） | POLIN + 起義 + 蕭邦三大博物館 |

**晚數分配**：WAW(1) + KRK(3) + WRO(1) + POZ(1) + WAW(1) = **7 晚**

### 為什麼修訂

| 修訂前的問題 | 修訂後的解法 |
|---|---|
| Day 1 抵達當天直奔克拉科夫 | Day 1 直接住華沙倒時差 |
| Day 2 時差未消當天去 Auschwitz | Day 3 已恢復狀態才去 |
| 鹽礦與 Auschwitz 兩個沉重景點疊加 | 改為 Auschwitz + 弗羅茨瓦夫 + 波茲南三城多元體驗 |
| 行李集中在最後一天搬 | 5 段火車分散搬運壓力 |

